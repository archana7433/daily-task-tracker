# Daily Task Tracker

A simple task manager web application built using **React.js** and **Flask (Python)**.

## Project Overview
This project allows users to add and manage daily tasks easily.  
It demonstrates the integration of a **React frontend** with a **Flask backend API**.

## Tech Stack

Frontend:
- React.js
- HTML
- CSS
- JavaScript

Backend:
- Python
- Flask

## Features

- Add new tasks
- View task list
- Simple and clean user interface
- Backend API using Flask

## How to Run the Project

### Backend

cd backend  
python app.py  

Server runs on:
http://127.0.0.1:5000

### Frontend

cd frontend  
npm start  

Frontend runs on:
http://localhost:3000
